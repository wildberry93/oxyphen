# oxyphen
Program for oxygen phenotype assignment based on proteome sequences

## Requirements
- Python2.7
- scikitlearn
- Biopython
- numpy
- pandas

## How to run oxyphen
Specify Blast, x and x pathways in SETTINGS file, eg. 

Run oxyphen with a proteome of choice as an input:
python oxyphen.py input_genome.fasta
